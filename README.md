# URROW Store

Магазин плагинов для Lampa.

## Установка

Настройки → Плагины → Добавить:
```
https://k1run9.github.io/urrowstore/store.js
```

## Что есть

- Меню слева → URROW Store
- Шапка → иконка 📦
- Настройки → URROW Store
- Пульт: вверх/вниз/влево/вправо/ОК/Назад
- Мышь: клик

## Как добавить плагин

Отредактируйте `sources.json`, запушьте — Actions обновит каталог.
